# Vendored Verso Blueprint documentation

These are verbatim copies of the authoring documentation from the
[verso-blueprint](https://github.com/leanprover/verso-blueprint) package,
vendored here so the FLT blueprint can be authored and built without a
separate checkout of that repository.

- `MANUAL.md` — the full Blueprint authoring and rendering reference
- `GETTING_STARTED.md` — the first-project walkthrough

Provenance: verso-blueprint branch `v4.30.0`, commit
`a1af8f3294b58cfb5b902c477b5d0429882b8454`. This matches the `VersoBlueprint`
pin in `../lakefile.lean`. Refresh these copies if that pin changes.

Note: internal links in these files (e.g. to `project_template/`) point at the
upstream verso-blueprint repository layout, not at this directory. For a worked
example in this repository, see `../FLTBlueprint/Chapters/Basics.lean`.
